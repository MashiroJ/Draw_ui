# AI绘画管理系统

一个基于Vue 3 + Element Plus的AI绘画管理系统前端项目。

## 功能特性

- 用户管理
  - 用户的增删改查
  - 用户状态管理
  - 用户角色分配

- 角色管理
  - 角色的增删改查
  - 角色状态管理
  - 角色权限分配

- 菜单管理
  - 菜单的增删改查
  - 菜单状态管理
  - 菜单图标配置
  - 菜单排序功能

- 画廊管理
  - 公共/私人画廊切换
  - 图片瀑布流展示
  - 图片点赞功能
  - 评论互动系统
  - 图片下载功能

## 技术栈

- Vue 3
- Vue Router
- Pinia
- Element Plus
- Axios
- Sass

## 开发环境

- Node.js >= 16
- npm >= 7

## 安装和运行

1. 克隆项目
```
git clone https://github.com/MashiroJ/Draw_ui.git

```

2. 安装依赖
```
npm install
```
3. 运行开发服务器
```
npm run dev
```

4. 构建生产版本
```
npm run build
```
